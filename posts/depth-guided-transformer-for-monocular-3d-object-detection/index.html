Depth-guided Transformer for Monocular 3D Object Detection

March 23, 2025

对于单目图像，目标深度估计是相当困难的，这是由于缺乏对真实场景空间的理解，进而检测的目标在图像坐标系下缺乏几何约束导致的。借助 LiDAR 数据与视觉信息融合可以很好的检测到目标的真实空间参数，那么借助 LiDAR 数据来训练一个可以从单目图像中预测伪深度图的深度预测器，并结合预测的伪深度图的深度特征和几何约束来理解单目图像空间信息，进而实现目标三维预测，这便是 MonoDETR 的设计思路。 该方法表明，即使是借助较为粗糙的伪深度前景图 DfgD_{fg}Dfg​ （其有监督标签为离散的分类深度图，并且深度只取目标中心点对应深度），添加感知深度特征 fDef_{D}^{e}fDe​ 的交叉注意力后，该 DETR 变体结构取得了最好的单目 3D 目标检测指标。

深度预测模块

该深度预测模块相当简单，可以看作就是对多尺度图像特征通过卷积采样，随后在特征图的每个位置上预测所有深度范围分类的归一化概率，深度图由所有位置上最大概率的深度范围分类对应的绝对深度值构成。值得注意的是，取 3×33\times 33×3 卷积后的图像特征作为深度图的深度特征。 前景深度图 DfgD_{fg}Dfg​ 的有监督学习过程如下所示： 训练好的深度预测模块相当于给模型提供了两个先验知识，图像中目标的深度信息特征以及目标伪深度图提供的几何约束。虽然深度特征就来自于训练伪深度图的过程中，但两者都参与了最终损失的计算，那么哪个先验知识对模型在 3D 目标检测任务上的提升贡献更大呢？

伪深度图深度特征

根据论文提供的深度注意力可视化热力图，解码器中的 Query 确实关注于目标范围内，但考虑到训练用的深度图真值只有目标框范围内有单一的值，背景信息在深度图中不存在意义，那么对应的注意力范围也应只在目标框范围上。因此可以假设深度感知交叉注意力模块的作用为，在包含所有目标框范围内根据不同的深度值感知目标位置。基于这个假设再来看上图，高注意的深度注意力范围存在两个问题，对相近深度的不同目标无法区分；不同目标的深度感知范围区别不大。这是否表明该深度感知 DETR 方法存在缺陷，值得进一步研究。

拉普拉斯损失函数

假设模型预测深度误差服从拉普拉斯分布（Laplace Distribution），也称为双指数分布，是一种连续概率分布。其特点是具有对称的“尖峰厚尾”形状，即中心位置有一个尖锐的峰值，而两侧尾部比高斯分布（正态分布）更厚重。这种分布在对异常值（离群点）敏感度较低的场景中非常有用，具有鲁棒回归（Robust Regression）的特性。 其概率密度函数为：

其中 μ\muμ 是位置参数（均值），bbb 是尺度参数，方差为 σ2=2b2\sigma^{2} = 2b^{2}σ2=2b2。 若要让预测深度尽可能接近真值，则取最小的其密度函数的负对数似然：
